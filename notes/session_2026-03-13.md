## scr/models/bd_models.py

```
class User(Base):
    __tablename__ = "users"
    id, full_name, email, hashed_password, created_at, role, is_active

class Course(Base):
    __tablename__ = "courses"
    id, title, description, creator_id → FK(users.id)

class Enrollment(Base):
    __tablename__ = "enrollments"
    user_id → FK(users.id) PK, course_id → FK(courses.id) PK, enrolled_at, is_completed

class Modul(Base):
    __tablename__ = "modules"
    id, title, order_index, course_id → FK(courses.id)

class Lesson(Base):
    __tablename__ = "lessons"
    id, title, content, module_id → FK(modules.id)

class MentorshipRequest(Base):
    __tablename__ = "mentorship_requests"
    id, sender_id → FK(users.id), mentor_id → FK(users.id), status
```

**Импорты:** `sqlalchemy.orm` — `Mapped`, `mapped_column` / `sqlalchemy` — `Integer`, `ForeignKey`, `func` / `datetime`
